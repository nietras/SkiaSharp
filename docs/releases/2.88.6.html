Version 2.88.6

Dependency & hardening release · Released · NuGet · GitHub Release

Highlights

This servicing release refreshes the bundled native image and text dependencies — WebP, HarfBuzz, and libjpeg-turbo — and restores no-SIMD builds needed for Microsoft Edge's enhanced security mode and Safari.

New Features

Engine

  • WebP 1.3.2 — Updates the bundled libwebp to 1.3.2. (#2623)
  • HarfBuzz 7.3.0 — Updates the bundled HarfBuzz to 7.3.0. (#2577)
  • libjpeg-turbo 3.0.0 — Updates the bundled libjpeg-turbo to 3.0.0. (#2581)

Security

  • Restore no-SIMD builds — Restores the no-SIMD native builds required by Microsoft Edge's enhanced security mode and Safari. (#2618)

Bug Fixes

  • Remove unused arm64e architecture — Drops the unused arm64e slice from native builds. (#2590)
  • Drop unused libjpeg — Removes the unused libjpeg dependency. (#2589)
