Version 3.0.0

Preview only · Superseded by 3.116.0 · Never released as stable — these changes rolled up into 3.116.0 · NuGet

Highlights

The first public preview of SkiaSharp 3.0 — a major modernization built on Skia m116, with new net8.0 (and early net9.0) targets, a modernized P/Invoke layer using function pointers and LibraryImport, and a large round of new drawing, picture, and shader APIs. As a preview that never shipped stable, all of this work rolled up cumulatively into 3.116.0. Many community contributors drove interop, API, and platform fixes across this preview.

⚠️ Breaking Changes

  • Removed all [Obsolete] members and types — Every previously deprecated API has been removed in 3.0. (#2539)
  • Modernized interop — The binding moves to function pointers and LibraryImport, changing the generated interop surface. ❤️ @maxkatz6 (#2917)
  • SKFilterQuality replaced by SamplingOptions — SKFilterQuality is superseded by sampling options, with compatibility overloads provided for migration. (#2789, #2924)

New Features

Engine

  • Skia m116 — Updates the bundled Skia engine to milestone 116. (#2829, #2885)
  • Further Skia update — ❤️ @Redth brings in additional Skia changes. (#3026)
  • Modernized repository — Reworks the repository and prepares the SkiaSharp 3.0 foundation. (#2505)

API Surface

  • Expose SKBlender — Adds bindings for SKBlender. (#2830)
  • More SKPicture APIs — Expands the SKPicture surface. (#2883)
  • R-Tree BeginRecording — Adds an R-Tree overload of BeginRecording. (#2889)
  • Skottie Animation Builder — Binds Skottie's Animation.Builder. (#2630)
  • SKImage.ToRawShader — Adds SKImage.ToRawShader. (#2748)
  • GetPixelSpan with offsets — Adds a GetPixelSpan() overload that accepts offsets. (#2609)
  • Span for SetRectRadii — ❤️ @Youssef1313 adds a Span overload for SetRectRadii. (#2949)
  • Spans for color filters — Adds Span overloads to the color filters. (#2879)
  • SKFilterQuality compatibility overloads — ❤️ @Youssef1313 adds compatibility overloads for SKFilterQuality. (#2963)
  • Trimmable PlatformConfiguration — ❤️ @maxkatz6 makes PlatformConfiguration properties trimmable. (#2717)

GPU & Rendering

  • SKGLView — Implements SKGLView. (#2598)
  • SKGLElement for WPF — ❤️ @gmurray81 adds SKGLElement to SkiaSharp.Views.WPF. (#2317)
  • WinUI accelerated views — Adds support for hardware-accelerated WinUI views. (#2733)
  • Metal on Mac Catalyst — Uses Metal as the backend on Mac Catalyst. (#2747)
  • Common Metal TFM — ❤️ @maxkatz6 brings the Metal APIs to a common .NET TFM. (#2788)

Text & Fonts

  • HarfBuzz 8.3.0 — Updates HarfBuzz to 8.3.0 (from 7.3.0 earlier in the cycle). (#2624, #2582)

Platform

  • .NET 8 — Updates the binding and samples to .NET 8. (#2927)
  • .NET 9 support — ❤️ @jeromelaban adds early support for .NET 9. (#3010)
  • WinForms GLControl — Uses the new GLControl packages for WinForms. (#2989)
  • Blazor component DPI — ❤️ @beto-rodriguez exposes DPI on the Blazor components. (#1832)

Images & Documents

  • libjpeg-turbo 3.0.0 — Updates libjpeg-turbo to 3.0.0 (later rolled back to 2.1.5.1 for stability). (#2583, #2699)
  • libwebp 1.3.2 — Updates libwebp to 1.3.2. (#2622)

Bug Fixes

  • MAUI memory leaks — Fixes memory leaks in the MAUI views. (#2955)
  • SKRectI rounding — Floors outwards when converting to SKRectI. (#2568)
  • Cloned SKPaint disposal — ❤️ @jeremy-visionaid fixes an AccessViolationException when disposing a cloned SKPaint. (#2904)
  • SKBitmap.Encode regression — ❤️ @sungaila reverts SKBitmap.Encode with SKPngEncoderOptions to the 2.88.8 behavior. (#3014)
  • Uno Skia pixel format — ❤️ @Youssef1313 fixes SKXamlCanvas on Uno Skia to use Bgra8888. (#2918)
  • XamlRoot null — ❤️ @jeromelaban handles a null XamlRoot when the SKXamlCanvas is unloaded. (#2854)
  • Metal snapshot color — ❤️ @taublast fixes Snapshot returning magenta on Apple Metal. (#2804)
  • Kerning adjustments — ❤️ @pdjonov fixes the GetKerningPairAdjustments API. (#2858)
  • async void in SKXamlCanvas — ❤️ @lindexi avoids async void in SKXamlCanvas. (#2720)
  • WASM canvas lookup — ❤️ @jeromelaban stops the WASM SKXamlCanvas failing when the canvas can't be found. (#2563)
  • Uno render context — ❤️ @jeromelaban ensures the canvas' context is active when rendering on Uno. (#2559)
  • nosimd builds — ❤️ @jeromelaban restores nosimd builds for enhanced-security Edge and Safari. (#2612)
