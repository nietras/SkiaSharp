Version 2.88.0

.NET MAUI & .NET 6 release · Released · NuGet · GitHub Release

Highlights

A landmark release that brings first-class .NET MAUI views (both compatibility shims and handlers), Mac Catalyst support, ASP.NET Blazor WebAssembly components, and full .NET 6 / net6.0 targets. The SkiaSharp package was also split into smaller per-platform "Native Asset" packages, and HarfBuzz was updated to 2.8.2. Several community contributors helped across Android, Uno, WebAssembly, and compliance.

New Features

API Surface

  • .NET MAUI views — Adds SkiaSharp.Views.Maui with both compatibility views and handlers. (#1678)
  • New drawing APIs — Surfaces additional Skia APIs to the managed binding. (#1828)

Text & Fonts

  • HarfBuzz 2.8.2 — Updates the bundled HarfBuzz to 2.8.2. (#1766)

Platform

  • Mac Catalyst — Adds Mac Catalyst support, moving build configuration into the gn files. (#1760, #1681)
  • ASP.NET Blazor WebAssembly — Adds support for Blazor WebAssembly components. (#1811)
  • .NET 6 targets — Adds net6.0-* targets for the binding projects. (#1707, #1740)
  • Xamarin.Forms on netcoreapp3.1 — Adds Xamarin.Forms support on .NET Core 3.1. (#1774)
  • WindowsAppSDK — Moves Windows support onto the WindowsAppSDK. (#1800)
  • Native Asset packages — Splits the SkiaSharp package into smaller per-platform "Native Asset" packages, with native Windows symbols and HarfBuzzSharp.NativeAssets.*. (#1758, #1797)
  • emscripten 2.0.23 — ❤️ @jeromelaban adds support for emscripten 2.0.23 on WebAssembly. (#1746)
  • Uno 4.0 — ❤️ @jeromelaban bumps the Uno Platform support to 4.0. (#1873)
  • Android packaging — ❤️ @jonathanpeppers stops shipping .so files in class libraries and trims Resource.designer.cs fields. (#1895, #1812)
  • Debug symbols — Adds the main PDB files to the packages. (#1781)

Security

  • Software Bill of Materials — ❤️ @mjbond-msft generates an SBOM manifest for the build. (#1954)

Bug Fixes

  • IgnorePixelScaling — Fixes the behavior of the IgnorePixelScaling property. (#1804)
  • Blazor SKCanvasView loop — ❤️ @JensKrumsieck prevents an infinite loop in the Blazor SKCanvasView. (#1889)
  • MAUI SKCanvasView NRE — ❤️ @jsuarezruiz guards against a NullReferenceException in the .NET MAUI SKCanvasView. (#1734)
  • Linux libdl resolution — ❤️ @akoeplinger tries libdl.so.2 before libdl.so. (#2010)
  • Windows handle locking — Replaces the HandleDictionary lock with a critical section on Windows. (#1817)
  • Blittable enums — Works around enums not being blittable. (#1857)
  • Native statics — Never disposes the native static objects. (#1863)
  • Mac Catalyst build — Fixes the Mac Catalyst build task Inputs/Outputs. (#1865)
