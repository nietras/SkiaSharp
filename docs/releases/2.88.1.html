Version 2.88.1

Skottie & platform release · Released · NuGet · GitHub Release

Highlights

This release introduces Skottie, SkiaSharp's Lottie animation engine, alongside new WinUI support and AVIF image encoding. It also refreshes the bundled native dependencies (libjpeg-turbo, libwebp, libexpat and zlib) for security and correctness. Skottie, WinUI and several fixes were driven by community contributors including @jeromelaban, @mgood7123, @koolkdev, @HarlanHugh and @RichardD2.

New Features

Engine

  • libjpeg-turbo 2.1.3 — Updates the bundled JPEG decoder. (#2206, skia mono/skia#97)
  • libwebp 1.2.3 — Updates the bundled WebP codec. (#2193)
  • libexpat 2.4.8 — Updates the bundled XML parser. (#2189)
  • zlib PNG fix — Updates zlib to resolve PNG loading issues. (#2045)

Animation

  • Skottie (Lottie) support — Adds a managed Skottie binding for playing Lottie/After Effects animations. Initial support landed from ❤️ @jeromelaban (#1987), with follow-up work adding nullable annotations and TimeSpan overloads, handling non-seekable streams, and moving Skottie and SceneGraph out of the core SkiaSharp.dll (#2091, #2119, #2126)

Images & Documents

  • AVIF encoding — Exposes the AVIF image encode format. ❤️ @mgood7123 (#2154)
  • Animations with a BOM — Supports loading animations whose data starts with a byte-order mark. (#2167)

API Surface

  • Writeable SKDataStream — Makes SKDataStream writeable, matching SKData. (#2128)
  • Nullable annotations & newer TFMs — Adds nullable reference type annotations and additional target frameworks across the libraries. (#2120)

Text & Fonts

  • SKShaper text alignment — SKShaper now honors the SKPaint.TextAlign property. ❤️ @koolkdev (#1910)

Platform

  • WinUI support — Adds a WinUI rendering backend. ❤️ @jeromelaban (#2042)
  • emscripten 3.1.7 — Updates the WebAssembly toolchain to emscripten 3.1.7. ❤️ @jeromelaban (#2094)
  • net6.0-tizen TFM — Adds a net6.0-tizen target framework. (#2099)

Bug Fixes

  • Fixed loading fonts with Unicode names — Resolves failures loading fonts whose file name contains Unicode text. (#2146)
  • Fixed Alpine builds — Corrects the Alpine Linux build. (#2168, #2192)
  • Guarded finalized locks — Checks whether a lock has been finalized before entering or leaving the critical section. ❤️ @RichardD2 (#2195)
  • Reshaped SKGLView on scale change — Reshapes the view when Window.BackingScaleFactor changes (#1853). ❤️ @HarlanHugh (#1854)
  • Returned null on invalid resize bounds — Avoids errors when resize bounds are invalid. (#2054)
  • Implemented a missing Mac/iOS feature — Fills in functionality that was previously unimplemented on Mac/iOS. (#2198)
  • Fixed Uno Roslyn generation — Forces Uno's Roslyn-hosted generators and adds the missing Uno.WinUI JavaScript support file. ❤️ @jeromelaban (#2199, #2086)

Plus the C API increment, newer .NET targeting and numerous CI and packaging improvements.
