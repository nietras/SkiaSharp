Version 3.116.0

SkiaSharp 3.0 — major release · Released · NuGet · GitHub Release

Supersedes 3.0.0 · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

Highlights

SkiaSharp 3.116.0 is the first stable release of the modernized SkiaSharp 3.0 line. It rolls up all of the preview-only 3.0.0 work cumulatively and ships it as a single stable package. The engine is updated to Skia m116, the interop layer is rebuilt on function pointers and LibraryImport, and the libraries now target .NET 8 and .NET 9 with multi-threaded/SIMD WebAssembly, a Metal backend on Mac Catalyst, and a wave of new span-based APIs. This is a major version with breaking changes — see below. Dozens of community contributors helped land this release, including @maxkatz6, @jeromelaban, @Youssef1313, @alexandrvslv, @Redth and @gmurray81.

⚠️ Breaking Changes

  • All [Obsolete] members and types removed — Every previously deprecated API has been deleted as part of the 3.0 cleanup. (#2539)
  • SKFilterQuality replaced by SamplingOptions — Filter quality is superseded by sampling options; compatibility overloads are provided to ease migration. (#2789, #2963, #2924)
  • Interop rebuilt on function pointers / LibraryImport — The P/Invoke layer now uses function pointers and source-generated LibraryImport, modernizing the native interop surface. (#2917)

New Features

Engine

  • Skia updated to m116 — Brings the bundled Skia engine up to milestone 116, plus follow-up external updates. (#2829, #2885) ❤️ @Redth (#3026)
  • HarfBuzz 7.3.0 → 8.3.0 — Updates the bundled HarfBuzz for improved text shaping. (#2582, #2624)
  • libjpeg-turbo 3.0.0 & libwebp 1.3.2 — Refreshes the bundled JPEG and WebP codecs. (#2583, #2622)
  • SkiaSharp 3.0 modernization — Modernizes the repository, build and packaging and targets .NET 8 and .NET 9. (#2505, #2927) ❤️ @jeromelaban (#3010)

GPU & Rendering

  • SKBlender — Exposes Skia's blender API for custom blend modes. (#2830)
  • SKImage.ToRawShader — Adds a raw-shader variant of image shaders. (#2748)
  • Metal backend on Mac Catalyst — Uses Metal as the rendering backend on Mac Catalyst. (#2747)
  • Common Metal TFM — Brings the Metal APIs to a common .NET target framework. ❤️ @maxkatz6 (#2788)
  • WinUI accelerated views — Adds support for hardware-accelerated WinUI views. (#2733)
  • Multi-threaded / SIMD WebAssembly — Adds support for multi-threaded and/or SIMD WebAssembly builds. (#2620)

API Surface

  • Span overloads across the API — Adds Span<SKPoint> overloads for SKMatrix.MapPoints/MapVectors and SKPath.AddPoly, a span overload for SetRectRadii, and span overloads on the color filters. ❤️ @alexandrvslv (#3030) · ❤️ @Youssef1313 (#2949) · (#2879)
  • GetPixelSpan() with offsets — Adds an offset-aware pixel span accessor. (#2609)
  • More SKPicture APIs & R-Tree recording — Expands the SKPicture surface and adds an R-Tree overload of BeginRecording. (#2883, #2889)
  • SKFilterQuality compatibility overloads — Provides compatibility overloads to bridge SKFilterQuality to the new sampling options. ❤️ @Youssef1313 (#2963)
  • Read-only GRMtlTextureInfo getters — Makes GRMtlTextureInfo getters read-only. (#2833)
  • Trimmable PlatformConfiguration — Marks PlatformConfiguration properties trimmable for smaller AOT/trimmed apps. ❤️ @maxkatz6 (#2717)

Text & Fonts

  • Skottie Animation Builder — Binds Skottie's animation builder for richer Lottie animation construction. (#2630)
  • Cleaner text-blob APIs — Tidies the SKTextBlobBuilder and SKRunBuffer APIs. (#2775)

Platform

  • SKGLElement for WPF — Adds an OpenGL-accelerated SKGLElement to SkiaSharp.Views.WPF. ❤️ @gmurray81 (#2317)
  • New WinForms GLControl — Moves WinForms onto the new GLControl packages. (#2989)
  • Blazor component DPI — Exposes the device pixel ratio on the Blazor components. ❤️ @beto-rodriguez (#1832)

Bug Fixes

  • Fixed MAUI view memory leaks — Resolves leaks in the MAUI views. (#2955)
  • Fixed access violation disposing a cloned SKPaint — Prevents an AVE when disposing a cloned paint. ❤️ @jeremy-visionaid (#2904)
  • Fixed magenta Metal snapshots — Corrects Snapshot returning a magenta image on Apple Metal. ❤️ @taublast (#2804)
  • Fixed Uno SKXamlCanvas color format — Uses Bgra8888 for SKXamlCanvas on Uno Skia. ❤️ @Youssef1313 (#2918)
  • Fixed null XamlRoot on unload — Handles XamlRoot being null when the SKXamlCanvas is unloaded. ❤️ @jeromelaban (#2854)
  • Avoided async void in SKXamlCanvas — Removes an async void path in SKXamlCanvas. ❤️ @lindexi (#2720)
  • Fixed GetKerningPairAdjustments — Corrects the kerning-pair adjustment API. ❤️ @pdjonov (#2858)
  • Restored SKBitmap.Encode behavior — Reverts SKBitmap.Encode with SKPngEncoderOptions to the previous (2.88.8) behavior. ❤️ @sungaila (#3014)
  • Fixed .NET 9 ASP.NET Blazor & WASM issues — Resolves Blazor base-path imports and .NET 9 Blazor/WebAssembly problems. (#3081, #3087, #3089, #3093)
  • Floored outwards converting to SKRectI — Rounds outward when converting to integer rects. (#2568)
  • Restored no-SIMD WebAssembly builds — Restores nosimd builds for hardened-security Edge and Safari. ❤️ @jeromelaban (#2612)

Plus the migration to 1ES pipelines, an MSVC build, dev-container support and many other CI, compliance and documentation improvements.
