Version 2.88.3

.NET 7 and WebAssembly improvements · Released · NuGet · GitHub Release

Highlights

This release adds .NET 7 support for Blazor views and the build infrastructure behind it, upgrades the WebAssembly build to Emscripten 3.1.12 with SIMD features, and improves GL interop for .NET. Community contributors @jeromelaban and @snnz drove the WebAssembly and Linux loader improvements.

New Features

GPU & Rendering

  • GL interception for .NET — Intercepts and captures GL for .NET. (#2268)

Platform

  • .NET 7 Blazor views — Adds .NET 7 support for Blazor views, along with the supporting build infrastructure. (#2254, #2255)
  • Emscripten 3.1.12 with SIMD — Generates for Emscripten 3.1.12 and enables SIMD features for WebAssembly. ❤️ @jeromelaban (#2259)
  • RTLD_DEEPBIND dlopen flag — Adds the RTLD_DEEPBIND flag to the dlopen mode on Linux. ❤️ @snnz (#2247)

Bug Fixes

  • WASM archive lookup — Fixes invalid globbing for the 3.1.12 archive lookup. ❤️ @jeromelaban (#2269)
  • Null-buffer skip — Treats a skip as a read into a null buffer. (#2265)
