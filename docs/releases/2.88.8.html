Version 2.88.8

Released April 10, 2024 · NuGet · GitHub Release

Highlights

A maintenance release that adds new method overloads for better API compatibility, hides OpenGL entry points that opengl32.dll does not export to prevent runtime errors on Windows, makes PlatformConfiguration properties trimmable, and removes an async void from SKXamlCanvas. Several fixes were backported via the @github-actions bot.

New Features

API Surface

  • New compatibility overloads — Additional method overloads to improve source compatibility. (#2810)

Bug Fixes

  • Windows OpenGL externs — No longer exposes OpenGL functions that opengl32.dll does not export, preventing EntryPointNotFoundException at runtime. (#2710)
  • SKXamlCanvas async void — Replaces async void with a safer async pattern to avoid unobserved exceptions. (#2731)
  • Trimming compatibility — Makes PlatformConfiguration properties trimmable, improving compatibility with trimmed and AOT builds. (#2734)

Platform Support

Platform What's New
🪟 Windows Unavailable opengl32.dll entry points are no longer surfaced
📦 General PlatformConfiguration properties are now trimmable
