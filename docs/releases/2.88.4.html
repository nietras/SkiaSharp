Version 2.88.4

Apple Silicon & WebAssembly release · Released · NuGet · GitHub Release

Highlights

This release adds native support for ARM64 (Apple Silicon) macOS machines and brings a wave of WebAssembly improvements — including exception handling, SIMD, and multithreading across .NET 7 and .NET 8 targets. It also refreshes core native dependencies for security and lands a large number of community-driven Uno, MAUI, and Tizen fixes.

New Features

Engine

  • Updated native dependencies — Bumps libpng to v1.6.40 and libexpat to 2.5.0+ for security and stability. (#2510)
  • Updated zlib to 1.2.13 — Refreshes the bundled zlib for the latest fixes. (#2484)

Platform

  • ARM64 macOS support — Adds support for Apple Silicon (ARM64) macOS machines. (#2468)
  • JSImport for WinUI on net7.0 — Moves the WASM WinUI target to JSImport. ❤️ @jeromelaban (#2428)
  • WASM exception handling & SIMD — Enables WebAssembly exception handling and SIMD in all configurations for net8+. ❤️ @jeromelaban (#2495)
  • WASM multithreading & SIMD — Adds mt and mt+simd variants and fixes HarfBuzz for emsdk 3.1.12. ❤️ @jeromelaban (#2286)
  • Blazor WASM assets for .NET 8 — Ships the new WASM assets in Blazor for .NET 8. (#2497)
  • Uno.WinUI target resolution — Disambiguates SKGLView for Uno and MAUI targets and adjusts net7.0 resolution for Uno.WinUI. ❤️ @jeromelaban (#2529, #2503)
  • Android .aar packaging — Ships the .aar with the Android NuGet. ❤️ @dellis1972 (#2465)
  • Hot Restart support — Includes the NativeReference for Hot Restart. (#2553)
  • .NET 8 Preview 4 support — Adds support for .NET 8 Preview 4. ❤️ @jeromelaban (#2457)
  • .NET iOS & Mac Catalyst samples — Adds a .NET iOS & Mac Catalyst sample and .NET "Core" sample support. (#2491)

API Surface

  • Faster SKColor hex parsing — Improves the performance of hex string parsing in SKColor. ❤️ @jwikberg (#2467)
  • Code cleanup — Reduces nesting levels using block-scoped using statements. ❤️ @Lehonti (#2528)

Security

  • Enabled CodeQL — Adds CodeQL static analysis to the build. ❤️ @Redth (#2303)
  • CVE scanning — Runs cve-bin-tool against native binaries. (#2490)

Bug Fixes

  • Fixed ToRect extension method — Corrects the ToRect extension. ❤️ @niza93 (#2392)
  • Fixed SKCanvasView dispose exception — Resolves an exception when the view is disposed before it has rendered. ❤️ @jjzhang12 (#2472)
  • Added guard against exception — Adds an additional check to avoid an exception. ❤️ @FoggyFinder (#2313)
  • Fixed Tizen canvas size calculation — Corrects the canvas size calculation on Tizen. ❤️ @myroot (#2322)
  • Corrected macOS Opaque handling — Uses the correct method for setting Opaque on macOS. (#2477)
  • Uno: SKXamlCanvas alpha type & Opaque — Updates the bitmap SKAlphaType in SKXamlCanvas.Wasm, defaults Opaque to false on macOS and iOS, and updates the SKSwapChainPanel for Android and iOS. ❤️ @roubachof (#2443, #2398, #2400, #2401)

Plus several CI, build, and tooling improvements, including pipeline fixes from @pjcollins (#2393, #2389) and a cgmanifest.json schema from @JamieMagee (#2232).
