Version 1.68.1.1

Bug-fix release · Released · NuGet · GitHub Release

Highlights

A focused patch release that corrects image decoding, hardens HarfBuzz string handling, and picks up upstream buffer-overflow fixes. Community contributors @bender2k14 and @daltonks landed fixes for sample rotation math and weak-reference cleanup.

New Features

Platform

  • buildTransitive .targets — Ships the package .targets file in the buildTransitive folder so it flows correctly to transitive consumers. (#1057)
  • Platform ref assemblies — Ensures the required assemblies are present in the platform reference folders. (#1044)

Security

  • Buffer-overflow fixes — Cherry-picks upstream fixes for buffer-overflow bugs. (#1037)

Bug Fixes

  • Correct image decoding — Fixes incorrect decoding of certain images. (#1055)
  • HarfBuzz strings — Resolves issues with HarfBuzz string handling. (#1034)
  • Weak-reference cleanup — ❤️ @daltonks makes TryRemove clean up instance WeakReferences correctly. (#1039)
  • Rotation matrix identity — ❤️ @bender2k14 starts the rotation matrix as the identity matrix in the animated sample. (#1062)
  • Sample cleanup — ❤️ @bender2k14 removes an unused TaskScheduler argument from AnimatedSampleBase. (#1063)
  • GPU sampling — Avoids going straight for the maximum sample count. (#1060)
