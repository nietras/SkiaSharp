Version 2.88.9

.NET 9 & platform fixes release · Released · NuGet · GitHub Release

Highlights

This release adds support for .NET 9 and ships a batch of WinUI/Uno SKXamlCanvas fixes, along with debug symbols now included in the packages. The WinUI native helper for GetByteBuffer was contributed by @jeromelaban.

New Features

Platform

  • .NET 9 support — Adds targeting and support for .NET 9. (#3012)
  • WinUI GetByteBuffer native helper — Adds a native helper enabling GetByteBuffer on WinUI. ❤️ @jeromelaban (#3039)

API Surface

  • Debug symbols in packages — The NuGet packages now ship with symbols for easier debugging. (#3046)

Bug Fixes

  • SKXamlCanvas on Uno Skia uses Bgra8888 — Corrects the pixel format used by SKXamlCanvas on Uno Skia. (#2919)
  • GetKerningPairAdjustments fix — Restores correct behaviour of the GetKerningPairAdjustments API. (#2886)
  • Null XamlRoot on unload — Guards against a null XamlRoot when an SKXamlCanvas is unloaded. (#2884)
