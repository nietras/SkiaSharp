Version 2.80.2

Bug-fix release with memory debugging and Uno GPU views · Released September 12, 2020 · NuGet · GitHub Release

Highlights

A stability-focused release fixing numerous bugs across image encoding, text rendering, interop, and native library loading. New memory debugging APIs help diagnose GPU resource usage, and the Uno Platform views gain GPU support plus WASM CPU rendering. Community contributor @ziriax drove two of the iOS and surface fixes.

New Features

Engine

  • Skia update — Bumped the native Skia engine to pick up upstream fixes (#1436).

API Surface

  • Memory debugging APIs — Exposed SKGraphics for diagnostics, plus additional GPU memory debugging members on GRContext (#1473, #1478).
  • SKColorF extension methods — New extensions to convert between native platform colors and SKColorF (#1469).
  • Replacement methods for obsoleted members — Added new methods to replace previously obsoleted APIs (#1431).

Platform

  • Uno Platform WASM support — CPU rendering views for WebAssembly via Uno Platform (#1333).
  • Uno Platform GPU views — New SKSwapChainPanel GPU view for Android, iOS, macOS, and WASM, plus further Uno integration (#1429, #1420).
  • Xamarin.Forms performance — Rendering performance improvements for Forms (#1424).
  • UWP non-blocking invalidation — UWP no longer blocks the UI thread when invalidating (#1468).
  • macOS redraw fix — Use NeedsDisplay instead of Display for correct macOS redraws (#1475).

Build & Packaging

  • Native-only NuGet package — A new package containing just the native binaries (#1456).
  • Debug bootstrapper builds — Passing --configuration=debug to the bootstrapper now produces a debug library (#1418).

Bug Fixes

  • SKSurface.Create origin ignored — Fixed the origin parameter being ignored when creating surfaces. ❤️ @ziriax (#1404)
  • SKCanvas.DrawTextOnPath null reference — Fixed a NullReferenceException in DrawTextOnPath on iOS. ❤️ @ziriax (#1410)
  • SKData from non-seekable stream — Fixed an exception when creating SKData from a non-seekable stream (#1411).
  • ASP.NET DllNotFoundException — Now checks the correct directory for native libraries under ASP.NET (#1483).
  • Image subset encoding — Fixed encoding overwriting the underlying pixels / encoding the full image instead of a subset (#1457).
  • WASM + GL — Use the correct native code for WASM with GL (#1445).
  • Transitive dependency native binaries — Native binaries now reach the final app via buildTransitive when only referenced transitively (#1440).
  • Text rendering — Use linear metrics for backwards compatibility and handle invalid text correctly (#1439, #1438).
  • Alpine no-dependency builds — Fixed Alpine builds that were still linking against fontconfig when asked not to (#1416).
  • Interop float return — Worked around cases where a float could not be returned correctly (#1453).
  • SetScaleTranslate revert — Reverted a regression in SetScaleTranslate (#1452).
