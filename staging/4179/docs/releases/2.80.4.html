Version 2.80.4

Servicing release · Released · NuGet · GitHub Release

Highlights

This servicing release fixes antialiasing edge behaviour and hardens surface allocation against failures, with a robustness fix contributed by @richirisu.

Bug Fixes

  • IsAntialias now controls edging — Setting IsAntialias on a paint now correctly drives the edging mode again. (#1802)
  • Graceful surface allocation failure — Returns an empty surface instead of faulting when the underlying allocation fails. ❤️ @richirisu (#1784)
  • ANGLE zlib1.dll output — The ANGLE build now also emits zlib1.dll, fixing a missing native dependency on Windows. (#1807)
